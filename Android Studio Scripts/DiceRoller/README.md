	# Dice Roller
***
## Introduction
This is an Android Application to Roll Dice.

***
## Features
* Rolls Dice.

***
## How to Use
1. Pull the code into your system.
2. Run the project in Android Studio.
3. Touch the Roll Button/Dice to roll the dice.
